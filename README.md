Coffee_bot Telegram-бот для кофейни

Что делает?
- Клиент авторизуется через Telegram WebApp
- Добавляет напитки в корзину, оформляет заказ
- Получает чек в Telegram
- Администратор управляет статусами заказов через отдельного бота

Где взять ключи?
- BOT_TOKEN: создать бота у @BotFather
- ADMIN_BOT_TOKEN: создать второго бота у @BotFather
- ADMIN_CHAT_ID: узнать у @userinfobot
- WEBAPP_URL: запустить ngrok http 3000 и скопировать ссылку

Запуск
-----------------------------------------------------------------------------------------------------
Вы должны создать 2 ботов:
1) Клиентский бот - где открывается Telegram WebApp для оформления заказа
2) Админ бот - с Админ бота, вы можете изменять статус заказа. Указывается номер заказа/имя клиента (здесь нет Telegram WebApp, все делается через чат)
3) Так же обязательно устанавливаем - ngrok
